# 华峰石化数字孪生平台 - UI功能需求文档

## 一、项目概述

| 项目 | 说明 |
|------|------|
| **项目名称** | 华峰石化数字孪生平台 |
| **技术栈** | HTML5 + CSS3 + JavaScript + Three.js |
| **设计风格** | 深蓝科技风，数字孪生可视化风格 |
| **核心功能** | 数据大屏展示、3D罐区模型、设备管理、视频监控、实景漫游、数据分析 |

---

## 二、系统架构

### 2.1 页面结构

```
┌─────────────────────────────────────────────────────────────────────┐
│  登录页面 (login.html)                                               │
│  └── 用户认证 → 跳转后台首页                                          │

│  后台管理系统 (带统一侧边栏导航)                                       │
│  ├── 首页 (index.html) - 数据概览与快捷入口                           │
│  ├── 罐区监控 (tank-monitoring.html) - 罐区实时监测                   │
│  ├── 设备管理 (equipment.html) - 设备台账与状态管理                    │
│  ├── 视频监控 (video.html) - 多路视频监控管理                         │
│  ├── 实景漫游 (roaming.html) - 3D场景自动/自由漫游                    │
│  ├── 数据分析 (analysis.html) - 数据统计与趋势分析                     │
│  ├── 数据大屏 (dashboard.html) - 全屏3D可视化大屏 [新开标签页]         │
│  └── 系统设置 (settings.html) - 系统配置与用户管理                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 五、页面功能详细需求

### 5.1 登录页面 (login.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **页面布局** | 左表单右视觉的左右分栏布局 (45%:55%) |
| **登录表单** | 用户名、密码输入框、记住密码、登录按钮 |
| **表单验证** | 非空验证、错误提示 |
| **视觉区域** | 渐变背景、Logo动画、平台标题、描述文字 |
| **登录逻辑** | 成功后跳转 index.html，失败显示错误信息 |
| **动画效果** | Logo浮动动画、背景光晕脉动 |

### 5.2 首页 (index.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **数据概览卡片** | 4个统计卡片横向排列：设备总数、正常运行、预警/告警、在线率 |
| **快捷入口** | 5个快捷入口：罐区监控、设备管理、视频监控、数据大屏、系统设置 |
| **罐区状态** | 各罐区设备状态分布可视化 |
| **实时告警** | 最近告警事件列表，带时间和状态标识 |
| **最近活动** | 系统操作日志时间轴 |
| **交互** | 卡片hover效果、快捷入口点击进入对应页面 |

### 5.3 罐区监控 (tank-monitoring.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **布局** | 左侧边栏(300px) + 右侧主内容区 |
| **罐区选择** | 下拉选择324/325/303罐区 |
| **设备列表** | 按罐区分类显示设备，支持点击选中 |
| **罐区可视化** | 3D/2D罐区场景展示 |
| **实时数据** | 温度、压力、液位实时数值显示 |
| **告警提示** | 异常设备红色高亮 |
| **趋势图表** | 24小时数据趋势曲线 |

### 5.4 设备管理 (equipment.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **布局** | 左侧设备树(280px) + 右侧设备详情 |
| **设备搜索** | 顶部搜索框，支持按名称搜索 |
| **设备树** | 按罐区分类展开/收起，显示设备类型图标 |
| **设备列表** | 表格展示：名称、类型、状态、位置、操作 |
| **状态筛选** | 全部/正常/预警/告警/离线筛选 |
| **批量操作** | 批量导出、批量删除 |
| **设备详情** | 点击行展开详情：基本信息、实时数据、历史记录 |
| **新增/编辑** | 弹窗表单编辑设备信息 |

### 5.5 视频监控 (video.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **布局** | 左侧摄像头列表(260px) + 右侧视频主区域 |
| **摄像头分组** | 按区域分组：球罐区、储罐区、泵房区、厂区入口 |
| **摄像头列表** | 显示摄像头名称、在线状态(绿点/红点) |
| **视频网格** | 支持1/4/9/16画面分屏显示 |
| **视频控制** | 播放/暂停、全屏、截图、录像 |
| **云台控制** | 方向控制、变焦、聚焦(模拟) |
| **选中交互** | 点击列表切换视频源 |

### 5.6 实景漫游 (roaming.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **布局** | 左侧控制面板(280px) + 右侧3D场景 |
| **漫游模式** | 自动漫游 / 自由漫游 两种模式卡片选择 |
| **漫游路径** | 预设路径：全罐区/324罐区/325罐区/303罐区 |
| **速度控制** | 1x/2x/4x 速度调节滑块 |
| **播放控制** | 开始/暂停/停止按钮 |
| **3D场景** | 3DGS模型加载、视角控制(旋转/平移/缩放) |
| **视角书签** | 保存/切换常用视角 |
| **全屏模式** | 沉浸全屏漫游 |

### 5.7 数据分析 (analysis.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **筛选条件** | 罐区选择、设备类型、时间范围(快捷选项/自定义) |
| **数据概览** | 顶部统计卡片：设备总数、在线率、告警数、平均运行时长 |
| **趋势分析** | 折线图：温度/压力/液位趋势对比 |
| **状态分布** | 饼图：设备状态占比 |
| **告警统计** | 柱状图：告警类型/时间分布 |
| **报表导出** | PDF/Excel格式导出 |
| **图表交互** | 数据缩放、图例筛选、tooltip提示 |

### 5.8 数据大屏 (dashboard.html) 【新开标签页】

| 功能模块 | 详细需求 |
|----------|----------|
| **整体布局** | 全屏沉浸式，无侧边栏，顶部弧形标题栏 |
| **顶部栏** | 平台标题 + 实时时间 |
| **左侧面板** | 罐区信息总览(4统计卡片)、安全监测指标(进度条)、实时告警事件列表 |
| **中心区域** | Three.js 3D罐区场景：中央控制室 + 8个储罐 + 管道连接 |
| **3D交互** | 鼠标拖拽旋转、滚轮缩放、悬停显示罐体数据、点击切换监控视频 |
| **右侧面板** | 设备状态总览(4统计卡片)、实时视频监控(单路)、环境参数监测 |
| **底部导航** | 能耗分析/园区总览/安防管理 切换标签 |
| **视觉效果** | 粒子效果、储罐发光动画、扫描线效果、科技感UI |
| **监控联动** | 点击3D场景中的罐体 → 右侧视频切换对应摄像头画面 |

**3D场景细节：**
- 8个储罐：A-001~A-006(球罐)、S-001~S-002(储罐)
- 状态颜色：蓝色(正常)、橙色(预警)、红色(告警)
- 每个罐体对应摄像头编号 CAM-01 ~ CAM-08
- 浮动标签显示：压力、温度、液位

### 5.9 系统设置 (settings.html)

| 功能模块 | 详细需求 |
|----------|----------|
| **布局** | 左侧设置导航(220px) + 右侧设置内容 |

| **通知设置** | 告警通知开关、通知方式(邮件/短信) |
| **用户管理** | 用户列表、新增/编辑/删除用户、权限分配 |
| **数据管理** | 数据备份、数据恢复、清空缓存 |

## 六、交互流程

### 6.1 用户登录流程

```
访问系统
    ↓
未登录 → 跳转 login.html
    ↓
输入用户名密码
    ↓
验证成功 → 保存登录状态 → 跳转 index.html
    ↓
验证失败 → 显示错误提示
```

### 6.2 数据大屏交互流程

```
进入 dashboard.html (新开标签页)
    ↓
加载Three.js 3D场景
    ↓
鼠标悬停罐体 → 显示浮动标签(压力/温度/液位)
    ↓
点击罐体 → 右侧视频区切换对应监控画面
    ↓
视频播放器显示：CAM编号、球罐名称、录制状态、实时时间
```

### 6.3 设备监控流程

```
进入 tank-monitoring.html
    ↓
选择罐区(324/325/303)
    ↓
左侧显示该罐区设备列表
    ↓
点击设备 → 右侧显示实时数据
    ↓
数据异常 → 红色高亮 + 告警提示
```

### 6.4 视频监控流程

```
进入 video.html
    ↓
左侧选择摄像头分组
    ↓
点击摄像头 → 右侧加载视频流
    ↓
视频控制：播放/暂停/全屏/截图
    ↓
分屏切换：1/4/9/16画面
```
